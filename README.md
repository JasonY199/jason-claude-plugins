# jason-claude-plugins

Personal Claude Code plugin marketplace.

## Plugins

| Plugin                                                                  | Description                                                    |
| ----------------------------------------------------------------------- | -------------------------------------------------------------- |
| [jason-handoff](https://github.com/JasonY199/jason-handoff)             | Session handoff skills and hooks for seamless context transfer |
| [jason-design-system](https://github.com/JasonY199/jason-design-system) | Design-system-first skill for UI development                   |

## Install

Add the marketplace, then install individual plugins:

```
/plugin marketplace add JasonY199/jason-claude-plugins
/plugin install jason-handoff@jason-claude-plugins
/plugin install jason-design-system@jason-claude-plugins
```

Or install plugins directly from their own repos:

```
/plugin marketplace add JasonY199/jason-handoff
/plugin install jason-handoff@jason-handoff

/plugin marketplace add JasonY199/jason-design-system
/plugin install jason-design-system@jason-design-system
```
